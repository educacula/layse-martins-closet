# Layse Martins Closet (PWA) — v2

App offline (PWA) para:
- Cadastro de produtos (SKU/código de barras), preço e **custo**
- Controle de estoque (entrada e venda)
- Financeiro com **lançamento automático** nas vendas (Receita + CMV)
- Relatórios (lucro 30 dias, gráfico de vendas por produto, gráfico financeiro)
- Backup em arquivo (salve no Google Drive)

## Upload no GitHub (GitHub Pages)
1. Abra seu repositório no GitHub (ex: `layse-martins-closet`)
2. Clique em **Code**
3. Clique em **Add file** → **Upload files**
4. Faça upload de **todos os arquivos e pastas** deste ZIP (index.html, sw.js, manifest.webmanifest e pasta assets)
5. Clique em **Commit changes**
6. Vá em **Settings** → **Pages**
7. Em **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main** / (root)
8. Abra o link do seu site (vai aparecer no topo do GitHub Pages)

## Instalar no Android (Moto G Stylus 5G 2024)
1. Abra o link do GitHub Pages no **Chrome**
2. Toque nos **3 pontinhos** → **Instalar app** / **Adicionar à tela inicial**
3. Abra o app pela tela inicial

## Atualizar da V1 para V2 (sem dor de cabeça)
1. Faça upload dos arquivos da V2 no GitHub (substituindo os antigos)
2. No celular:
   - Feche o app (tire da multitarefa)
   - Abra de novo
3. Se ainda ficar a versão antiga:
   - Chrome → **Configurações** → **Privacidade e segurança** → **Limpar dados de navegação** (marque Cache)
   - Ou desinstale o app e instale de novo

> Como o app usa cache (PWA), às vezes precisa 1–2 aberturas para atualizar.
